## <img src="https://img.shields.io/badge/▸-ff00ff?style=flat-square&label=&labelColor=0d1117" /> 核心能力 · Core Expertise

### ⚡ Trace 调试与崩溃分析

> **调试信条**：遇到崩溃，先开 Trace，而不是加 `printf`。

| 技术点 | 说明 |
|:---|:---|
| **HardFault / MemManage / BusFault / UsageFault** | 从 `CFSR` `HFSR` `MMFAR` `BFAR` 寄存器定位故障源 |
| **CoreSight Trace** | DWT 周期计数与性能测量 · ITM 软件跟踪 · ETM 指令流追踪 |
| **Segger RTT / SystemView** | 实时日志输出与任务调度可视化，零侵入调试 |
| **栈回溯 (Stack Backtrace)** | 通过 LR、SP、EXC_RETURN 重建调用链 |
| **Post-mortem 分析** | 从故障现场寄存器、栈内存、Core Dump 还原崩溃过程 |

### ⚡ 可靠性与固件升级
- **Bootloader / OTA**：A/B 分区 · 非活跃分区烧录 · CRC16/32 双重校验 · 回滚确认
- **故障保护**：IWDG · WWDG · 多级故障降级策略

### ⚡ 工业通信协议栈
- **EtherCAT 从站**：LAN9253 PDI 中断优化 · SM 邮箱/PDO 映射 · 1ms 周期 · DWT 时序诊断
- **CAN-FD 自定义协议**：29-bit 扩展 ID 寻址 · DLC 32 控制帧 · Little-endian 线格式
