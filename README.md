## <img src="https://img.shields.io/badge/◉-00b4d8?style=flat-square&label=&labelColor=0d1117" /> 核心能力 · Core Expertise

### ◆ Trace 调试与崩溃分析

> 遇到崩溃，先开 Trace，而不是加 `printf`。

| 能力 | 详情 |
|:---|:---|
| **Fault 分析** | HardFault / MemManage / BusFault / UsageFault 定位，从 `CFSR` `HFSR` `MMFAR` `BFAR` 还原现场 |
| **CoreSight Trace** | DWT 周期计数 · ITM 软件跟踪 · ETM 指令流追踪 |
| **Segger 工具链** | RTT 实时日志 · SystemView 任务调度可视化 |
| **栈回溯** | 通过 LR、SP、EXC_RETURN 重建调用链 |
| **Post-mortem** | 故障寄存器 · 栈内存 · Core Dump 现场还原 |

### ◆ 可靠性与固件升级
- **Bootloader / OTA**：A/B 分区 · 非活跃分区烧录 · CRC16/32 校验 · 回滚确认
- **故障保护**：IWDG · WWDG · 多级故障降级

### ◆ 工业通信
- **EtherCAT 从站**：LAN9253 PDI 中断优化 · SM 邮箱/PDO 映射 · 1ms 周期 · DWT 时序诊断
- **CAN-FD 协议**：29-bit 扩展 ID · DLC 32 控制帧 · Little-endian 线格式
