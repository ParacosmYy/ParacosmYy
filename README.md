## About Me

- 强项在软件，硬件懂一点，够用来做开发和联调。
- 目前重点放在 IMU、MCU/SoC 联合开发、工业通信和产品交付。
- 接触并使用过 `FreeRTOS`、`RT-Thread`，也在持续了解 `Zephyr`、`ThreadX`、`uC/OS`。
- 代码风格偏向解决问题、推进业务，而不是单纯堆概念。

## Current Focus

| 方向 | 说明 |
| --- | --- |
| `EtherCAT + CiA402` | 继续深入工业通信与运动控制协议栈 |
| `IMU` | 着手推进传感器数据链路、采集与应用开发 |
| `MCU -> SoC -> MCU OTA` | 关注端到端升级链路与可靠交付 |
| `HPM` | 正在推进高性能平台上的嵌入式实践 |
| `RTOS` | 已用 `FreeRTOS` / `RT-Thread`，持续扩展 `Zephyr` / `ThreadX` / `uC/OS` |

## Project Highlights

- `CAN-FD 透传 EtherCAT 模块`
  - 打通工业总线与上位链路的协议转换与透传场景。
- `机器人小脑 / PMS 散热 / 关节模组通信`
  - 面向机器人子系统的通信、控制与热管理协同开发。
- `PC -> SoC -> MCU OTA 升级`
  - 从上位机到 SoC 再到 MCU 的完整升级链路实践。
- `IMU 开发`
  - 正在推进传感器链路相关的底层接入与业务应用。
